# 🚀 DevScope – Developer Intelligence Platform

DevScope is a powerful platform that analyzes GitHub profiles and provides deep insights like developer score, comparison, and resume generation.

---

## 🔥 Features

- 📊 Developer Score based on GitHub activity  
- 🔍 GitHub Profile Analysis  
- ⚔️ Developer Comparison  
- 📄 Auto Resume Generator  
- 📈 Interactive Charts & Stats  

---

## 🛠 Tech Stack

- Frontend: React.js  
- Backend: Node.js, Express  
- API: GitHub REST API  
- Charts: Chart.js / Recharts
